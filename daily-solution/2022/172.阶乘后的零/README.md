# 172. 阶乘后的零
给定一个整数 n ，返回 n! 结果中尾随零的数量。

提示 n! = n * (n - 1) * (n - 2) * ... * 3 * 2 * 1

 

示例 1：

输入：n = 3
输出：0
解释：3! = 6 ，不含尾随 0
示例 2：

输入：n = 5
输出：1
解释：5! = 120 ，有一个尾随 0
示例 3：

输入：n = 0
输出：0
 

提示：

0 <= n <= 104
 

进阶：你可以设计并实现对数时间复杂度的算法来解决此问题吗？

## 题解
### 我的题解

```python
class Solution:
    def trailingZeroes(self, n: int) -> int:
        cnt=0
        # 先把其中有5因数(不止含一个)的数拿出来5，10，15...
        # 符合条件的数的数量为(n//5)+1
        for fam5 in range(1, (n//5)+1):
            # 还原该数
            muti5 = 5 * fam5
            # 解剖含多个5因数的数，把5都取出来，直到不再含有5因数
            while muti5%5==0:
                muti5//=5
                cnt+=1
        return cnt+1
```
